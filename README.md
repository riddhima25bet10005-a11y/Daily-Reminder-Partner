# Daily Reminder Partner
A smart, customizable daily reminder application that helps you stay organized and productive throughout your day.
# Features
•	Smart Reminders: Set one-time or recurring reminders for your daily tasks
•	Multiple Categories: Organize reminders by work, personal, health, etc.
•	Cross-Platform: Available on web, mobile, and desktop
•	Snooze Options: Flexible snooze durations for when you need extra time
•	Priority Levels: Mark reminders as low, medium, or high priority
•	Natural Language Processing: Set reminders using natural language (e.g., "Remind me to call John in 2 hours")
•	Cloud Sync: Your reminders sync across all your devices
•	Custom Sounds: Choose from various notification sounds
•	Dark/Light Mode: Choose your preferred theme


# Usage
1.	Create a Reminder:
o	Tap the "+" button
o	Enter your reminder text
o	Set date/time or use natural language
o	Choose category and priority
o	Save
2.	Manage Reminders:
o	Swipe left to delete
o	Tap to edit
o	Long press to mark as completed
3.	Settings:
o	Adjust notification preferences
o	Set default snooze duration
o	Configure backup options
